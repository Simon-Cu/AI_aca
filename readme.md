# AI Academic Chart Assistant

一个面向学术研究场景的 AI 多智能体助手，用于理解学术图表、检索相关文献、评估文献相关性，并生成结构化中文研究报告。

项目基于 FastAPI、LangGraph 与 OpenAI 兼容的大语言模型构建，提供网页界面和流式 API。

## 功能

- 上传并解析学术图表
- 提取图表类型、方法、指标、数据集和趋势等可见信息
- 根据用户问题和图表内容自动生成学术检索词
- 通过 OpenAlex 检索相关论文
- 使用 Tavily 进行网页补充检索
- 对候选论文进行相关性和证据强度评估
- 根据任务类型自动编排多智能体工作流
- 支持图表解读、文献调研，以及两者结合的任务
- 保存本地对话线程与任务状态
- 提供浏览器可用的聊天界面和 Swagger API 文档
